# Security Policy

Please do not open public issues for security vulnerabilities.

Report security concerns by emailing: hirreshsundra3@gmail.com

SemZero may process dbt manifests, compiled SQL, table names, model names, lineage metadata, and optional warehouse metadata. Do not include production secrets, credentials, private customer data, or sensitive query output in bug reports.

## Supported status

SemZero is currently `0.8.0-alpha`. Use shadow/advisory mode first and review findings manually before enabling stricter policy behavior.
