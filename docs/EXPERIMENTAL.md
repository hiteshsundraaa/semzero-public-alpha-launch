# Experimental / Legacy SemZero Surfaces

The public 0.8.0-alpha product core is the **dbt PR Assumption Gate**.

These modules are retained because they contain useful research and future-product ideas, but they are not the first-time user path:

- full Wind Tunnel / warehouse replay
- Chaos Mode
- streaming shadow gate
- GNN / RGCN graph experiments
- broad premerge validation packs
- repair automation
- Slack and broad platform integrations

Promotion rule: a module only moves into the core when it improves hidden-assumption extraction, assumption validation, blast radius, receipts, PR review, calibration, cost savings, or noise control for dbt PRs.
