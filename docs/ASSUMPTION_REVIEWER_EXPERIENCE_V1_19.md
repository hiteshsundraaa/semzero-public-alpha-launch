# SemZero Core v1.19 — Reviewer Experience

This release keeps SemZero core-only and non-blocking. It changes the PR comment from a long evidence dump into a reviewer-first summary grouped by action:

- Must review
- Useful advisory
- Accepted risk / exceptions
- Needs feedback

Full evidence remains in the JSON receipt. The comment is intentionally capped to reduce review noise and help engineers decide what to check first.
